# **SQL Practice Projects – Northwind & World Databases**

This repository contains SQL exercises and analytical queries using the **Northwind** and **World** sample databases.  
It includes a structured workbook of tasks and solutions covering business-oriented analysis (Northwind) and global geographic/demographic analysis (World).  
All queries are written for learning, practice, and portfolio demonstration.

---

## **Included Databases**

### **1. Northwind**
A business-focused dataset simulating a trading company.  
Contains customers, orders, products, employees, suppliers, and shipping data.

Used for:
- Sales analysis  
- Customer and order reporting  
- Supplier and product insights  
- Inventory queries  
- Employee and shipper performance  

### **2. World**
A geographic and demographic dataset containing countries, cities, and languages.

Used for:
- Population analysis  
- Regional statistics  
- Language distribution  
- Country/city comparisons  
- Continental and demographic trends  

---

## **What This Repo Contains**

- A workbook outlining tasks for both Northwind and World  
- Individual `.sql` files grouped by topic  
- Beginner to intermediate SQL queries  
- Realistic analysis scenarios  
- Clean, readable, and reusable SQL code

---

## **Skills Demonstrated**

Across both databases, this project covers:

- `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL JOIN`  
- Aggregations (`SUM`, `AVG`, `COUNT`, `MAX`, `MIN`)  
- Grouping & filtering (`GROUP BY`, `HAVING`, `WHERE`)  
- Subqueries (scalar, correlated, nested)  
- Sorting and limiting (`ORDER BY`, `LIMIT`)  
- Data cleaning logic  
- Business and demographic analytics  
- Creating views and calculated fields  
- Window functions (where supported)

## **How to Use**

1. Import the **Northwind** and **World** databases into MySQL (or compatible SQL engine).  
2. Open the `.sql` files from this repo.  
3. Run each query independently—no setup required.  
4. Use the workbook as a guided exercise set.
